Código fonte

O comando document.getElementByI('teste') é escrito em linguagem JavaScript, aqui envelopamos o comando com a tag code

Organizando

A tag code não organiza o seu código, deixa tudo um do lado do outro. Por isso, precisamos envelopar a tag code com a tag pre

Com code

Aqui usamos um código Python para exemplificar

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') print ('Fim do programa')

Com code + pre

        
num = int(input('Digite um número'))
if num % 2 == 0:
print(f'O número {num} é PAR')
print ('Fim do programa')
